箭头函数是 PHP 7.4 的新语法，是一种更简洁的 匿名函数 写法。

匿名函数和箭头函数都是 Closure 类的实现。

箭头函数支持与 匿名函数 相同的功能，只是其父作用域的变量总是自动的。
